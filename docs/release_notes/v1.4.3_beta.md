# Aries AI v1.4.3 （预发布版）发布说明
**更新时间**：2026年4月9日

感谢您使用 Aries AI。这不仅仅是一次普通的迭代，更是我们产品向更智能、更成熟的阶段迈出的重要一步——本次灰度测试/预发布版本着重优化了与 AI 进行复杂对话时的流式输出体验，并引入了全新的“用户体验改进计划”框架。

## 重磅更新内容

**1. 引入全新「用户体验改进计划」（UX Program）**
我们在关于界面的设计中，新增了一个名为“用户体验改进计划”的功能选项，默认关闭。若您同意参与，我们将收集小部分非身份识别的基础设备日志与报错信息来快速诊断、优化产品。这对于提升预发布测试版（以及未来的正式版）稳定性至关重要。
- **调整点**：新的《隐私政策与用户协议》说明。我们已对隐私声明做出对应章节的正式调整与扩充，以保障使用清晰及合规。

**2. 核心架构：流式思考内容深度重构 (Stream WebView Render)**
为了让 AI 在思考深层逻辑或输出复杂 Markdown 时有完美的格式排版，此版本针对后台的流处理链路 (`StreamRenderHelper`) 和组件进行重做。
- **特性**：解决上一版本中某些特定模型响应过程中偶现“思考框过早异常收缩”及“输出文本乱码不渲染”的致命故障。新解析器稳定截获 `<think>` 标签并按预设展开，在答案正文到达时再驱动 WebView 高效解析视图。

## 优化与修复
- 改进 `AboutActivity.kt`，新增了平滑动画的 UX Switch 选项。
- 修复了其他几处潜在崩溃引起的问题。
